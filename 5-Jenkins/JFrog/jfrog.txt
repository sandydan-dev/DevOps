- why need Artifactory tool

  - share link of Artifactory.

  - store end-product to Github, JFrog, Nexus

# Jfrog.com  > free trial

> Create maven repository in Jfrog

- go to User Menu > quick repository creation 
  - select package type: maven  > next 
  - repository prefix : sandyjfrog
  - create 

  - create token then add to the Jenkins
    - go to Adminsitration  (tab)  
    - inside UserManagement > Access Token 
    - generate token 

    - Description : jenkins-access-tokey  (any-name)
    - token scope : Admin 
    - username : admin  (any-name)
    - generate token 

   (copy token, kept this token to jenkins) 
   (connect JFrog to Jenkins)


> Go to Jenkins > Manage Jenkins 
  - Plugins : Artifactory (tool installing)   
  - go back to jenkins

> manage Jenkins > Credentials
- stores scoped to Jenkins > Global > Add Credentials
  - new Credentials
  - kind: Username with password
  - username : admin 
  - password : <jfrog-token/password>
  - ID  : artifact-cred 
  - Description : artifact-cred 
  - create
