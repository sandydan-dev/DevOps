âœ… Features of Tomcat

1: Servlet and JSP Support 
- Its core features > execute java servlet and JSP Pages 

2: Lightweight and Fast 
- Small footprint, quick to install, easy to configure

3: Support HTTPS/SSL 
- Tomcat can handle secure communication with certificates.

4: WAR file Deployment
- you can deploy apps by simple copying ".war" file to the "webpages/" directory. 

5: Automatic Application Reloadind
- If files change, Tomcat can reload the app automatically, 

6: Built-in manager and host manager
- Web UI to: 
  - start/stop application 
  - deploy/undeploy apps 
  - check server health


7: Logging and Monitoring
- access logs
- server logs 
- application logs 
- performance monitoring 

8: Scalability
- can handle high traffic using:
  - clustering
  - load balancing 
  - session replication
