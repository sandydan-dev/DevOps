AWS Security Group

It is a firewall which is stand front of instance, and manage traffic (inbound / outbound)

An AWS Security Group (SG)  is like a virtual firewall that controls INBOUND (comming inside) and
OUTBOUND (going outside) traffic for your EC2 Instance 


Act as virtual firewall    =  Works at EC2 instance level
Stateful                   = if you allow incoming traffic, return traffic is automatically allowd 
Inbound rules              = Control traffic comming into your instance
Ountbound rules            = control traffic going out from your instance
Allow only                 = you cannot deny, you can only ALLOW ports/IPs
Default Security Group     = Block all inbound, allows all outbound
