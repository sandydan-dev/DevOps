AWS instance

Install Nginx Server 

login as root user 
cmd: sudo su -

cmd: yum update -y   # for amazon linux
cmd: apt update -y   # for ubuntu linux

install nginx server 
cmd: sudo apt install nginx -y    # for ubuntu 
cmd: yum install nginx -y         # for amazon-linux 

then start the nginx server and enable to auto start

cmd: sudo systemctl start nginx
cmd: sudo systemctl enable nginx

go to folder inside nginx path
cmd: cd /var/www/html 
result : index.html     # this is main index file will run on web server 


take aws instance publicIP and and paste into new tab and see web content
